# 毕业设计答辩讲稿

**题目：基于双通道时序信号的两阶段管道泄漏检测与定位方法**

各位老师好，我是孟达，我的毕业设计题目是《基于双通道时序信号的两阶段管道泄漏检测与定位方法》。本课题主要围绕管道泄漏场景下的自动识别与距离估计问题展开，目标是利用左右双通道传感信号，实现从“是否发生泄漏”到“泄漏位置估计”的完整分析流程。

首先介绍研究背景。管道输送系统广泛应用于能源、化工和市政等领域，一旦发生泄漏，可能造成介质损失、设备损伤，甚至引发环境和安全事故。传统人工巡检方式难以满足连续监测和快速定位的需求，而基于传感信号的数据驱动方法能够提供更及时的状态判断。本文关注的是泄漏声信号在管道中的传播特征。泄漏发生后，流体喷射会引起湍流、压力脉动和管壁振动，这些扰动会沿管道传播，并在左右两个传感器上形成幅值、相位和到达响应的差异。因此，双通道信号不仅可以用于判断泄漏状态，也为进一步估计泄漏距离提供了依据。

在数据处理方面，本文从原始 CSV 时序数据出发，将连续信号切分为固定长度的 5 秒窗口。由于采样率为 25.6 kHz，每个窗口包含 128000 个采样点，属于长时序信号。为了避免相邻片段同时出现在训练集和测试集中导致指标虚高，本文采用文件级划分策略，即同一个原始文件切分出的所有片段只进入同一个数据子集。最终构建了两个任务数据集：Stage2 分类任务包含 11704 个单通道样本，Stage1 回归任务包含 3737 个双通道配对样本。

本文的核心方法是两阶段框架。第一阶段是 Stage2 分类任务，输入单通道分段信号，输出正常状态和不同泄漏形态的类别，用于判断是否存在泄漏。第二阶段是 Stage1 回归任务，只针对泄漏样本，输入左右双通道配对信号，输出泄漏距离。这样设计的原因是，正常样本没有距离标签，如果把分类和距离回归强行混在一起，会造成标签语义不清。两阶段拆分之后，分类和定位可以分别优化，也更符合实际工程中“先报警、再定位”的流程。

在模型设计方面，本文首先采用一维卷积神经网络作为基础模型。1D CNN 通过多层卷积和逐步降采样，从长时序波形中提取局部冲击、持续能量变化和多尺度模式。随后，本文引入 Conformer 结构进行对比。Conformer 结合卷积和自注意力机制，在保留局部建模能力的同时增强长程依赖表达，用于分析更复杂时序结构是否能带来提升。评价体系方面，分类任务主要使用 Accuracy、Macro-F1 和混淆矩阵；回归任务使用 MAE、RMSE、Bias、P95 误差和不同距离档位下的误差表现。同时，本文同时报告片段级和文件级结果，使评测更接近连续采集场景。

实验结果表明，Stage2 分类任务表现较好。1D CNN 在片段级测试集上取得 Accuracy 0.9415、Macro-F1 0.9180，文件级 Accuracy 达到 0.9783。Conformer 在分类任务上进一步提升，片段级 Accuracy 达到 0.9900，Macro-F1 达到 0.9883，文件级指标达到 1.0000。混淆矩阵显示，片段级仍存在部分类别间混淆，但文件级聚合后误判明显减少，说明分类结果在连续采集场景下较稳定。

Stage1 距离回归任务中，1D CNN 取得片段级 MAE 1.90 米、RMSE 2.43 米，说明双通道原始波形中确实包含可学习的距离信息。但从误差分布看，部分距离档位误差仍较大，例如少样本档位和边界距离更容易出现偏差，模型存在一定向中间距离收缩的现象。此外，传统 SVM 基线在回归任务上取得 MAE 1.51 米，优于 1D CNN，说明在当前数据规模下，手工时频特征和传统机器学习方法仍有竞争力。通道消融实验进一步表明，单通道输入会使 MAE 从 1.90 米恶化到 3.88 米，证明左右通道差异是定位任务的关键信息来源。

综上，本文的主要贡献包括：第一，构建了从原始双通道 CSV 信号到固定窗口训练样本的数据处理流程，并采用文件级划分提高评测可信度；第二，建立了 Stage2 分类和 Stage1 回归的两阶段泄漏检测定位框架；第三，对比了 1D CNN、Conformer、传统机器学习基线和通道消融实验，分析了不同方法的适用边界。当前工作的不足主要在于数据规模和工况覆盖仍有限，部分距离档位样本不均衡，回归任务的 P95 误差仍偏大，跨工况泛化和在线部署还需要进一步验证。后续可以从数据扩充、时频特征融合、距离感知损失和更合理的文件级聚合策略等方向继续改进。

我的汇报到此结束，恳请各位老师批评指正。
