# 答辩可能提问与实验基本原理

本文档用于答辩前准备，分为两部分：

1. 老师可能会问的问题及参考回答
2. 本实验中常见指标、模型和实验设计的基本原理

## 一、老师可能提问与参考回答

### 1. 为什么要把任务拆成两阶段，而不是直接训练一个模型同时分类和定位？

参考回答：

因为分类和定位的标签语义不同。分类任务中包含正常样本和泄漏样本，但正常样本没有泄漏距离标签。如果把正常样本也放进距离回归任务中，就会出现标签语义混杂。

所以本文将任务拆成两个阶段：Stage2 先判断信号状态，回答“是否泄漏、属于哪类状态”；Stage1 只在泄漏样本上进行距离回归，回答“泄漏大概在哪里”。这种设计更符合工程监测流程，也便于分别分析分类错误和定位误差。

### 2. 为什么你的 PPT 中 Stage1 和 Stage2 的顺序看起来不是先 1 后 2？

参考回答：

这里的 Stage1 和 Stage2 是按照实验脚本或任务模块命名，不完全等同于实际部署流程中的先后顺序。实际工程流程应该是先进行状态分类，确认存在泄漏后，再调用距离回归模型进行定位。

因此在汇报中我会明确说明：逻辑流程是“先分类、后定位”，Stage1 和 Stage2 是本文实验中两个任务模块的名称。

### 3. 为什么分类任务使用单通道，而回归任务使用双通道？

参考回答：

分类任务主要依赖局部波形形态、能量变化和泄漏状态特征，单通道中已经包含较明显的判别信息，所以可以完成状态识别。

而定位任务更依赖左右传感器响应差异，例如幅值差、相位差、到达时间差和整体波形对应关系。单个通道只能看到局部响应，很难判断泄漏点相对于左右传感器的位置。因此回归任务使用双通道输入更符合定位问题的物理机制。

通道消融实验也验证了这一点：双通道 CNN 的 MAE 为 1.90 米，而单通道 MAE 恶化到 3.88 米，说明双通道信息对定位非常关键。

### 4. 为什么要采用文件级划分，而不是随机划分所有片段？

参考回答：

因为同一个原始采集文件切分出的相邻 5 秒片段高度相似。如果随机划分，来自同一次采集的片段可能同时出现在训练集和测试集中，模型可能记住文件背景特征，而不是真正学到泄漏规律。

文件级划分保证同一个原始文件的所有片段只属于一个数据集，可以减少数据泄漏风险。虽然这样评测更严格，但更接近模型面对新采集文件时的真实表现。

### 5. 为什么文件级分类结果比片段级更好？

参考回答：

单个 5 秒片段可能受到噪声、瞬态波动或局部异常影响，因此片段级会出现偶然误判。文件级聚合相当于利用同一采集文件中的多个片段进行综合判断，例如投票或平均，可以抵消部分偶然误差。

所以文件级结果更稳定，也更接近实际监测系统中“连续一段时间内判断状态”的方式。

### 6. 为什么文件级回归结果没有明显优于片段级回归？

参考回答：

分类错误往往有一定随机性，多个片段投票后容易被抵消。但回归误差可能包含系统性偏差，例如某些距离档位整体被预测到中间区域。

如果每个片段都存在相似方向的偏差，简单平均不会消除误差，反而可能保留甚至放大这种偏差。因此 Stage1 文件级回归没有像分类那样明显提升，说明后续需要设计更合理的聚合策略，比如基于置信度加权、异常片段筛选或输出不确定性。

### 7. 为什么 Conformer 分类效果更好，但回归不一定更好？

参考回答：

Conformer 结合了卷积和自注意力，能同时建模局部波形和长程上下文，因此对分类任务有帮助。分类主要判断整体状态，长程信息可以增强类别区分。

但回归任务更依赖左右通道之间细粒度差异，例如相位关系、到达时间差和幅值差。模型更复杂不一定能更好保留这些细粒度信息，特别是在样本数量有限时，还可能出现训练不稳定或过拟合。因此 Conformer 对分类有效，但不必然对距离回归更优。

### 8. 为什么 SVM 在回归任务中比 CNN 更好？

参考回答：

这说明深度模型并不是在所有任务上都一定最优。当前回归样本数量相对有限，而 SVM 在小样本和结构化特征场景下通常具有较好的泛化能力。

另外，如果传统特征中已经包含较强的距离相关信息，SVM 可以比较稳定地拟合这种关系。CNN 虽然能够从原始波形中自动学习特征，但需要更多数据支撑。因此本文保留传统基线的意义，就是客观判断深度模型是否真正带来额外收益。

### 9. 为什么决策树表现相对弱？

参考回答：

决策树通过一系列特征阈值进行划分，模型表达方式比较离散。对于高维时序信号或连续距离回归问题，单棵决策树容易对训练数据过拟合，同时泛化能力有限。

相比之下，SVM 和 KNN 对小样本特征空间关系的利用更平滑，CNN 则可以从波形中学习层级特征，所以决策树整体表现相对较弱是可以理解的。

### 10. 为什么距离回归误差在不同距离档位差异很大？

参考回答：

主要原因可能有三个。

第一，不同距离档位样本数量不均衡。少样本档位训练不足，模型更容易向样本更多的距离区域预测。

第二，边界位置本身更难预测。模型容易出现向中间区域收缩的现象，导致边界距离误差较大。

第三，不同位置的声传播和传感器响应差异不一定线性，某些位置的信号特征可能更相似，导致模型混淆。

因此按距离档位分析比只看平均 MAE 更有诊断价值。

### 11. 你的定位 MAE 是 1.90 米，这个结果在工程上够用吗？

参考回答：

目前还不能直接认为已经达到工程精确定位要求。MAE 1.90 米说明双通道波形中存在可学习的距离信息，模型能够估计大致位置；但 P95 误差接近 5 米，说明尾部误差仍然较大。

所以本文更准确的结论是：定位任务已经验证可行，但距离回归鲁棒性仍需提升。后续需要补充困难档位数据、引入物理约束、改进文件级聚合，并进行跨工况验证。

### 12. 为什么要报告 P95 误差？

参考回答：

MAE 反映平均误差，但平均值可能掩盖少数大误差。工程应用中，少数很大的定位误差也可能导致排查范围扩大，影响维修效率。

P95 误差表示 95% 样本的误差不超过该值，可以反映尾部风险。本文报告 P95，是为了说明模型不仅要看平均表现，也要关注极端错误。

### 13. Bias 为 +0.45 米是什么意思？

参考回答：

Bias 表示预测值相对于真实值的平均偏差。Bias 为正，说明模型整体上略微高估距离；Bias 为负，则说明整体低估距离。

本文 Bias 为 +0.45 米，说明模型平均预测距离比真实距离大约偏高 0.45 米。这个偏差不算特别大，但说明回归模型仍存在一定系统误差。

### 14. 为什么单通道也能分类，但不能很好定位？

参考回答：

分类和定位需要的信息不同。分类只需要判断信号是否具有泄漏特征，以及属于哪种状态，单通道中通常已经有足够的波形变化。

定位则需要判断泄漏点相对左右传感器的位置，这需要比较两个通道之间的响应差异。单通道缺少参照关系，因此定位能力明显下降。

### 15. 你的模型是否能用于真实管道？

参考回答：

目前本文主要完成受控实验环境下的流程验证，还不能直接说明可以泛化到所有真实管道。真实工程中还会受到压力、流量、管径、材料、传感器安装方式、环境噪声和工况变化影响。

因此本文结论应限定为：在当前实验数据条件下，两阶段框架可行，双通道信号包含可学习的距离信息。要用于真实管道，还需要跨工况、跨批次和在线部署验证。

### 16. 如果后续继续做，你最优先改进什么？

参考回答：

我会优先改进 Stage1 距离回归。具体包括：补充误差较大的距离档位数据；尝试时频融合特征；引入距离感知损失或物理传播约束；改进文件级聚合策略；并输出预测置信区间。

因为分类任务已经较稳定，而定位误差是当前限制工程应用的主要问题。

### 17. 你的创新点是什么？

参考回答：

本文的创新点不在于提出一个全新的复杂网络结构，而在于针对双通道泄漏信号建立了一套比较完整且可信的实验流程。

具体包括：第一，构建原始 CSV 到固定窗口样本的数据处理流程；第二，采用文件级划分降低数据泄漏风险；第三，将泄漏识别和距离定位拆成两阶段任务，避免标签语义混杂；第四，通过深度模型、传统基线、文件级聚合和通道消融共同验证分类与定位能力。

### 18. 为什么不用传统互相关方法直接定位？

参考回答：

传统互相关方法通常依赖传播速度、到达时间差和较清晰的信号峰值。在复杂噪声或波形不稳定情况下，互相关峰值可能不明显，定位效果会受影响。

本文使用深度学习和机器学习方法，是希望模型能够从原始波形或特征中自动学习距离相关模式。当然，传统互相关方法仍然有很强物理解释性，后续可以作为基线或与模型融合。

### 19. 数据标准化为什么要按片段独立进行？

参考回答：

按片段标准化可以减少不同采集文件之间幅值尺度差异的影响，使模型更关注波形形态、相对变化和通道差异。

但这种做法也有一个边界：如果绝对幅值本身包含重要泄漏强度或距离信息，独立标准化可能会削弱这部分信息。因此后续也可以比较全局标准化、文件级标准化和片段级标准化对结果的影响。

### 20. 你的实验中最能支撑结论的证据是哪几个？

参考回答：

我认为有四组证据最关键。

第一，文件级划分结果说明评测流程比较可信。第二，Stage2 分类结果说明状态识别已经较稳定。第三，Stage1 回归 MAE 说明双通道信号中存在可学习的距离信息。第四，通道消融显示单通道误差明显恶化，证明定位依赖左右通道差异。

这四组结果共同支撑本文结论：分类成熟，定位可行但仍需提升。

## 二、实验基本原理

### 1. 分类任务与回归任务的区别

分类任务的输出是离散类别。例如本文 Stage2 输出正常状态或不同泄漏状态，模型关心的是预测类别是否正确。

回归任务的输出是连续数值。例如本文 Stage1 输出泄漏距离，模型关心的是预测值和真实值之间相差多少米。

因此，分类任务常用 Accuracy、Precision、Recall、F1 等指标；回归任务常用 MAE、RMSE、Bias 和误差分布等指标。

### 2. Accuracy 准确率

Accuracy 表示所有样本中预测正确的比例。

公式为：

```text
Accuracy = 预测正确的样本数 / 总样本数
```

例如 100 个样本中预测对了 94 个，Accuracy 就是 94%。

Accuracy 简单直观，但如果类别不均衡，可能会掩盖问题。例如 90% 样本都是正常类，模型全部预测正常，也能得到 90% 准确率，但对泄漏类完全没有识别能力。

### 3. Precision 精确率

Precision 表示模型预测为某一类的样本中，有多少是真的这一类。

公式为：

```text
Precision = TP / (TP + FP)
```

其中 TP 是真正例，FP 是假正例。

以“泄漏类”为例，如果模型预测 100 个样本为泄漏，其中 90 个确实泄漏，10 个其实正常，那么 Precision 就是 90%。

Precision 关注的是“模型报出来的结果有多准”。

### 4. Recall 召回率

Recall 表示真实属于某一类的样本中，有多少被模型找出来了。

公式为：

```text
Recall = TP / (TP + FN)
```

其中 FN 是假负例。

以“泄漏类”为例，如果真实有 100 个泄漏样本，模型找出了其中 85 个，那么 Recall 就是 85%。

Recall 关注的是“真实存在的问题有没有被漏掉”。

### 5. F1 分数

F1 是 Precision 和 Recall 的调和平均。

公式为：

```text
F1 = 2 * Precision * Recall / (Precision + Recall)
```

F1 同时考虑“报得准不准”和“漏得多不多”。当 Precision 和 Recall 都高时，F1 才会高；如果其中一个很低，F1 也会被拉低。

在泄漏检测中，F1 比单独看 Accuracy 更有意义，因为它能更好地反映模型对不同类别的综合识别能力。

### 6. Macro-F1

Macro-F1 是多分类任务中常用的指标。它的做法是：先分别计算每个类别的 F1，然后对所有类别的 F1 取平均。

公式可以理解为：

```text
Macro-F1 = 每个类别 F1 的平均值
```

例如五分类任务中，先分别计算正常类、泄漏类型 1、泄漏类型 2、泄漏类型 3、泄漏类型 4 的 F1，然后五个数相加再除以 5。

Macro-F1 的特点是每个类别权重相同，不会因为某一类样本数量多就占更大权重。因此，它适合用于类别不均衡的数据集。

答辩时可以这样解释：

“Accuracy 看整体预测正确率，Macro-F1 更关注每一类是否都识别得比较均衡。如果数据类别不均衡，只看 Accuracy 可能会被多数类掩盖，所以本文同时报告 Accuracy 和 Macro-F1。”

### 7. Micro-F1 与 Macro-F1 的区别

Micro-F1 是先把所有类别的 TP、FP、FN 汇总，再计算整体 F1。样本多的类别对结果影响更大。

Macro-F1 是先算每个类别的 F1，再平均。每个类别权重相同。

简单理解：

```text
Micro-F1 更看整体样本表现
Macro-F1 更看各类别是否均衡
```

本文使用 Macro-F1，是因为希望避免某些样本较多的类别掩盖少数泄漏状态的识别问题。

### 8. 混淆矩阵

混淆矩阵用于观察分类模型具体把哪些类别预测对了、哪些类别混淆了。

矩阵的行通常表示真实类别，列表示预测类别。对角线上的数值表示预测正确，非对角线表示误判。

通过混淆矩阵可以发现：模型是否容易把某两类混淆，是否某一类经常被漏判，或者是否模型偏向某一类。

在本文中，片段级混淆矩阵显示局部误判主要集中在相近泄漏形态之间；文件级聚合后误判明显减少，说明连续片段决策更稳定。

### 9. MAE 平均绝对误差

MAE 是回归任务中最直观的指标，表示预测值和真实值之间绝对误差的平均值。

公式为：

```text
MAE = |预测值 - 真实值| 的平均值
```

例如真实距离是 3 米，预测为 5 米，误差就是 2 米。对所有样本的绝对误差取平均，就是 MAE。

本文 CNN 回归片段级 MAE 为 1.90 米，意思是平均每个片段的距离预测误差约为 1.90 米。

### 10. RMSE 均方根误差

RMSE 是先计算误差平方的平均值，再开方。

公式为：

```text
RMSE = sqrt((预测值 - 真实值)^2 的平均值)
```

RMSE 对大误差更加敏感。因为误差会先平方，大误差会被放大。

如果 RMSE 明显大于 MAE，通常说明存在一些较大的预测错误。本文同时报告 MAE 和 RMSE，是为了既看平均误差，也观察大误差风险。

### 11. Bias 平均偏差

Bias 表示预测值相对于真实值的平均有符号误差。

公式为：

```text
Bias = (预测值 - 真实值) 的平均值
```

MAE 会取绝对值，因此看不到模型是高估还是低估。Bias 不取绝对值，可以观察系统性偏差。

如果 Bias 为正，说明模型整体偏高估；如果 Bias 为负，说明模型整体偏低估。本文 Bias 为 +0.45 米，表示模型平均预测距离略高于真实距离。

### 12. P95 误差

P95 误差表示 95% 样本的误差都不超过这个数值。

例如 P95 误差为 4.99 米，可以理解为在测试样本中，95% 的预测误差不超过约 4.99 米，剩下 5% 样本误差可能更大。

P95 用来观察尾部风险。工程应用不能只看平均误差，因为少数很大的误差也可能造成严重影响。

### 13. 容差命中率

容差命中率表示预测误差落在某个容许范围内的比例。

例如 1 米容差命中率表示：

```text
|预测距离 - 真实距离| <= 1 米 的样本比例
```

这个指标更接近工程需求，因为实际定位中常常关心“有多少样本能落在可接受误差范围内”。

本文按距离档位统计 1 米以内命中率，可以观察哪些距离位置预测稳定，哪些位置预测困难。

### 14. 文件级聚合

文件级聚合是指把同一个原始采集文件切出来的多个片段结果合并成一个文件级结果。

对于分类任务，可以使用投票或概率平均。比如一个文件切成多个片段，大多数片段预测为某一类，则文件级结果就是该类。

对于回归任务，可以对多个片段的预测距离取平均或中位数。

文件级聚合的意义是减少单个片段的偶然误判，更接近实际连续监测场景。但如果回归误差存在系统偏差，简单平均不一定改善结果。

### 15. 1D CNN 的基本原理

1D CNN 是一维卷积神经网络，适合处理时间序列信号。卷积核沿时间轴滑动，提取局部波形特征。

在本文中，1D CNN 可以学习泄漏信号中的局部冲击、能量变化和多尺度波形模式。多层卷积逐步扩大感受野，使模型能够从短时间局部特征扩展到更长时间范围的整体特征。

相比手工设计特征，CNN 的优势是可以直接从原始波形中自动学习特征；相比更复杂的模型，CNN 结构更稳定、训练成本更低，适合作为强基线。

### 16. Conformer 的基本原理

Conformer 是一种结合卷积和自注意力的模型结构。

卷积模块擅长提取局部模式，例如局部波形变化和短时冲击；自注意力模块擅长建模长程依赖，可以让模型关注序列中相距较远但相关的片段。

因此，Conformer 理论上比纯 CNN 更适合同时建模局部和全局信息。本文结果也显示，Conformer 在分类任务中表现很好。但对于回归任务，复杂模型不一定更优，因为距离估计更依赖双通道细粒度关系和足够的数据支撑。

### 17. SVM 的基本原理

SVM，即支持向量机。分类时，它的核心思想是在特征空间中寻找一个最优分隔边界，使不同类别之间的间隔尽可能大。

用于回归时，通常称为 SVR。它的目标是在允许一定误差范围的情况下，找到一个尽量平滑的函数来拟合数据。

SVM/SVR 在小样本和特征维度较高的场景中常常表现稳定。因此本文中 SVM 回归效果较好，说明传统特征中可能已经包含较强距离信息，同时当前数据规模还不足以让深度模型充分发挥优势。

### 18. KNN 的基本原理

KNN，即 K 近邻方法。它的思想很直观：对于一个新样本，在训练集中找到与它最相似的 K 个样本，然后根据这些邻居的标签进行预测。

分类时，KNN 通常采用多数投票；回归时，KNN 通常取邻居目标值的平均。

KNN 的优点是简单直观，不需要复杂训练；缺点是依赖特征距离度量，如果特征空间不能很好反映真实相似性，效果会受影响。

### 19. 决策树的基本原理

决策树通过一系列“如果某个特征小于某个阈值，就走左分支，否则走右分支”的规则进行预测。

它的优点是可解释性强，容易理解；缺点是单棵树容易过拟合，尤其在高维复杂数据中泛化能力有限。

本文中决策树表现相对较弱，说明泄漏分类和距离回归并不是简单依靠几个阈值就能稳定解决的问题。

### 20. 通道消融实验的意义

消融实验是指有意去掉某个模块或信息，观察模型性能变化，从而判断这个模块或信息是否重要。

本文的通道消融是将双通道输入改为单通道输入，比较定位误差变化。结果显示单通道 MAE 明显恶化，说明左右通道差异是距离估计的关键信息来源。

答辩时可以强调：

“通道消融不是为了提高指标，而是为了证明双通道设计的必要性。”

### 21. 为什么定位比分类难？

分类只需要判断信号属于哪个离散状态，类别边界相对清晰时，模型容易取得较高准确率。

定位是连续数值预测，需要模型学习距离和波形变化之间的细粒度关系。不同距离之间信号变化可能很微弱，还会受到噪声、工况和传感器响应影响。

此外，回归任务对样本分布更敏感。如果某些距离档位样本少，模型就容易向样本多的区域预测。因此定位通常比分类更难。

### 22. 本文实验结论应该如何准确表述？

建议答辩时不要说：

```text
本文已经实现了高精度工程定位。
```

更稳妥的说法是：

```text
本文建立了基于双通道时序信号的两阶段泄漏识别与定位流程。实验表明，分类任务已经较稳定，双通道信号中存在可学习的距离信息，定位任务具有可行性。但当前距离回归误差仍受数据覆盖、档位分布和物理约束不足影响，后续需要进一步提升鲁棒性。
```

这个表述既能体现工作价值，也不会把结论说得过满。
