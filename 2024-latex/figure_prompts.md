# 论文补图 AI 绘图 Prompt 清单

以下 prompt 用于后续生成论文插图。建议统一风格：白底、学术论文信息图、蓝灰主色、少量红色高亮泄漏点、中文标签、无照片质感、无复杂背景、横向 16:9 或 4:3 构图，适合插入本科毕业论文。

**建议位置：** 第二章“管道泄漏探测基本原理”，替换当前图 2-2。

**Prompt：**

绘制一张中文学术论文风格的信息图，主题为“管道泄漏声信号传播与双通道响应差异”。画面中央是一条水平管道，管道中部偏左有一个红色泄漏点，泄漏点向外喷射并产生声波/压力波。管道左右两端分别放置“左端传感器”和“右端传感器”，用蓝色波纹箭头表示泄漏声信号沿管道向两侧传播。左侧路径较短、响应更强，右侧路径较长、响应衰减更明显。图下方增加三个简洁模块：“传播距离差异”“幅值衰减差异”“到达时间/相位差异”，并用箭头指向“距离估计依据”。整体白底，蓝灰配色，泄漏点用红色强调，线条清晰，中文标签，适合 LaTeX 论文插图，不要三维渲染，不要照片风格。
